git命令初步认知与使用

  • git

我司一直用的SVN,对Git的需求也不是很大;出于个人兴趣吧想简单学习一下git,避免自身碰到git的一些命令或问题时像个门外汉一样一窍不通;

安装git

https://git-scm.com/downloads 选择合适的系统文件下载;

安装完成后git --version查看是否安装成功

初始化本地仓库

定位到需要创建的文件位置 git init
如下图所示创建完成后会在相应的地方看到.git隐藏文件;若要去除与远程仓库的连接删除(不建议进行删除) .git文件夹

代码提交

连接远程仓库 git remote add origin [远程仓库地址] origin为github默认的连接名

git add [所要提交的文件夹名]

git commit -m "提交的注释"

git push origin [分支名]

克隆远程库

git clone [远程仓库地址]

git clone -b [指定分支名] [仓库地址] [本地目录]
