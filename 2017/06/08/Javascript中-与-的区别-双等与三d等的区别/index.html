Javascript中= = 与 = = = 的区别(双等与三d等的区别)

  • js

Integer,String,double等基础数据类型中= =与= = =是有区别的

==默认会进行一次数据类型转换再进行值比较
===不进行类型转换若类型不一至则直接返回false

如下图所示:

ArrayList,Map,json等高级数据类开中= =与= = =是没有区别的

是进行指针地址比较

基础类型与高级类型，= =和= = =是有区别的

1）对于==，将高级转化为基础类型，进行“值”比较
2）因为类型不同，===结果为false
