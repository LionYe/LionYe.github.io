linux 新磁盘分区、挂载、软链接

  • liunx

由于服务器资源有限，出来了根目录盘被100%;新加一块磁盘并将某些容易涨数据的目录软链接过去解决根本问题；

一、新硬盘2 100GB

二、分区、格式化、挂载
fdisk -l 查看磁盘信息发现了新增的100G磁盘sdb

fdisk /dev/sdb 进行分区
n 输入n 分区
p 输入P分为主分区(e为逻辑分区)
1 输入1即为第一个主分区即为sdb1（输入2会有sdb2、输入3会有sdb3以此类推）
回车（默认） First Cylinder是选择该分区的起始磁盘数，这里可自定义也可不做选择，默认是1，如无特殊需求强烈建议选择默认，也就是1来分区（直接按回车）
回车（默认） 接下来是定义该分区的大小，如果按默认（按回车）即是使用全部可用存储额，也可以是用M或m单位结尾的数字(大写M是大B的意思，如果输入1M实际上是X8也就是8m的空间)

W 保存
fdisk -l 查看的新磁盘分区已存在

mkfs -t ext3 -c /dev/sdb1 进行格式化，如有多个分区可把sdb1改成sdb2 sdb3…以此类推，具体可用fdisk -l看到每个分区的名字

mkdir mnt 新建mnt目录
mount /dev/sdb1 /mnt 将分区sdb1挂载到/mnt 目录下
df -TH 可以看到成功挂载

vi /etc/fstab 编辑/ect/fstab ；新增挂载的信息

三、软链接
ln -s 源文件 目标文件 目标文件需不存在（若已存在、对其进行备份然后删除、软链接后进行还原）

Example:
ln -s /mnt /logs 将/logs数据存在/mnt下

/logs会变成一个快捷键指向/mnt目录
